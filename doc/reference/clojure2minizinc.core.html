minizinc

(minizinc mzn & {:keys [solver mznfile print-mzn? num-solutions all-solutions?], :or {solver *fd-solver*, mznfile (doto (java.io.File/createTempFile "clojure2minizinc" ".mzn") .deleteOnExit), print-mzn? false, num-solutions 1, all-solutions? false}})

Calls a MiniZinc solver on a given MiniZinc program and returns a list of one or more solutions.

Options are

:mzn (string) a MiniZinc program, which can be created with other functions of clojure2minizinc :solver (string) solver to call :mznfile (string or file) MiniZinc file generated :print-mzn? (boolean) whether or not to print resulting MiniZinc program (for debugging)

Solver options :num-solutions (int) An upper bound on the number of solutions to output :all-solutions (boolean) If true, return all solutions
